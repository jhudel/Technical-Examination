Setting Up and Testing the Application Locally

Prerequisites:

Before you begin, make sure you have the following software installed on your machine:

.NET Core SDK
SQL Server
Step 1: Set up locally the application

Step 2: Configure the Database Connection

Open the appsettings.json file located in the extracted folder using a text editor of your choice.

In the ConnectionStrings section, modify the connection string to point to your local SQL Server instance.

Step 3: Apply Database Migrations

Open a command prompt or terminal in the application's directory (the folder where you extracted the application).

Run the following command to apply database migrations:
dotnet ef database update

This command will create the necessary database tables based on the application's data model.

Step 4: Run the Application

Open a command prompt or terminal in the application's directory.

For Back end side, run the following command to start the application locally:
-dotnet run

For Front end side, run the following command to start the application locally
-npm start

Step 5: Access the Application

Open a web browser (e.g., Chrome, Firefox, or Edge).

In the browser's address bar, enter the following URL:
http://localhost:4200
